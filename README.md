## ✨ Features

### 🎨 Design
- **Dark Professional Theme** — Deep navy/blue palette with vibrant cyan accents
- **Glass Morphism** — Modern frosted glass effects on navigation and cards
- **Responsive** — Pixel-perfect on mobile, tablet, and desktop
- **Smooth Animations** — Scroll-triggered fade-ins, parallax, 3D tilt effects
- **Premium Typography** — Inter & JetBrains Mono from Google Fonts

### 📄 Sections
- **Hero** — Animated typewriter effect, floating particles, gradient orbs
- **About Me** — Bio, stats, contact info cards
- **Skills** — Filterable categories with animated progress bars
- **Projects** — Featured cards with 3D tilt hover, award badges
- **Achievements** — Interactive timeline with scroll animations
- **Certifications** — Grid showcase of all certifications
- **Education** — Visual timeline of academic journey
- **Resume** — Embedded ATS-friendly resume with print/PDF export
- **Contact** — Contact form with validation, info cards, social links
- **Footer** — Quick links, social icons, copyright

### 📋 ATS Resume
- Clean, semantic HTML structure
- Print-optimized layout (`Ctrl+P` to save as PDF)
- Keyword-rich content targeting 95+ ATS score
- Proper heading hierarchy (H1 → H2 → H3)

### 🎮 Easter Eggs
- Konami Code (`↑↑↓↓←→←→BA`) — try it!
- Styled console messages

## 🚀 Getting Started

### Option 1: Open Directly
Simply open `index.html` in your browser.

### Option 2: Live Server (VS Code)
1. Install the "Live Server" extension in VS Code
2. Right-click `index.html` → "Open with Live Server"

### Option 3: GitHub Pages
1. Push to a GitHub repository
2. Go to Settings → Pages
3. Select `main` branch and root folder
4. Your site will be live at `https://username.github.io/repo-name/`

---

## 📥 Resume Download
- Navigate to the **Resume** section on the website
- Click **"Print / Save as PDF"**
- In the print dialog, select **"Save as PDF"**
- The ATS-friendly resume will be exported with clean formatting
